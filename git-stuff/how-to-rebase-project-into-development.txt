1. In intelliJ command line, enter "git fetch -a" in console
2. Pull Git → Rebase → development
3. unix>mvn clean package -Pprod (Since you are not building the RDA, no -PbuildRda)
4. After that is done, unix>git push -f (to force push changes)

IF NOTHING HAPPENS

5. unix>git branch -d development (to remove the local development branch)
6. unix>git checkout -f development (to recreate the development branch from GitLab)
7. unix>git checkout -f <desired branch name> (To switch back to your branch)
8. unix>git push -f to (force push changes)
